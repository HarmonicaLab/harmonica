# 🎶 Harmonica Lab

**Professional tools for diatonic harmonica players.**

Live: [harmonicalab.github.io/harmonica](https://harmonicalab.github.io/harmonica)

---

## Tools

### 🎵 Harmonica Lab Tool
Interactive note chart, pitch tuner, metronome and difficulty levels for diatonic harmonica.
→ [harmonicalab.github.io/harmonica](https://harmonicalab.github.io/harmonica)

### 📝 Harmonica Lab Tab Transcriber
Real-time pitch detection and tablature builder for diatonic harmonica.
Supports all 12 keys, blow/draw/bend detection, session saving and PDF export.
→ [harmonicalab.github.io/harmonica/tabscriber](https://harmonicalab.github.io/harmonica/tabscriber)

---

## Features

- 🎙️ Real-time pitch detection (YIN algorithm)
- 🎼 All 12 diatonic harmonica keys (C, D, E, F, G, A, Bb, Ab, F#, Low F, Low D, Low C)
- 🎵 Blow, draw, draw bend and blow bend detection
- 📝 Tablature builder with undo/redo and session saving
- 📄 PDF export
- 🌙 Light/dark theme
- 🇮🇹 / 🇬🇧 Italian and English interface
- 📱 Mobile and desktop friendly
